# โครงงาน: การวิเคราะห์ข้อมูลสถิติการท่องเที่ยวภายในประเทศ ปี 2567

## วัตถุประสงค์
เพื่อวิเคราะห์แนวโน้มของนักท่องเที่ยวในแต่ละภูมิภาคของประเทศไทย
และสร้างการแสดงผลข้อมูลในรูปแบบภาพ เพื่อสนับสนุนการบริหารจัดการด้านการท่องเที่ยวและการตัดสินใจเชิงกลยุทธ์

## เครื่องมือที่ใช้
- Python (Pandas, Matplotlib)
- Google Colab
- Looker Studio
- Microsoft Excel
- Canva (สำหรับจัดทำสไลด์นำเสนอ)

## ขั้นตอนการดำเนินงาน
1. เก็บรวบรวมข้อมูลจากเว็บไซต์การท่องเที่ยวแห่งประเทศไทย (ททท.)
2. ทำความสะอาดข้อมูลด้วย Microsoft Excel และแปลงเป็นไฟล์ .CSV
3. วิเคราะห์และสร้าง Visualization ด้วย Python (Colab)
4. นำเสนอข้อมูลผ่าน Looker Studio และสไลด์ Canva

## ผลลัพธ์ที่ได้
- แสดงแนวโน้มจำนวนนักท่องเที่ยวในแต่ละภูมิภาคของประเทศไทย ปี 2567  
- สร้างแผนภูมิและแดชบอร์ดสรุปข้อมูลจำนวนผู้เข้าพักและการเดินทาง  
- สรุป insight เพื่อสนับสนุนแนวทางการพัฒนาอุตสาหกรรมท่องเที่ยว  

## หน้าที่ของผู้จัดทำ
- เขียนโค้ดใน Google Colab ด้วย Python (Pandas, Matplotlib)  
- ทำความสะอาดและจัดเตรียมข้อมูลใน Excel  
- สร้าง Visualization และ Dashboard ใน Looker Studio  
- จัดทำรายงานและสไลด์นำเสนอใน Canva  

## สิ่งที่ได้เรียนรู้
- เข้าใจกระบวนการวิเคราะห์ข้อมูลจริงตั้งแต่ต้นจนจบ  
- พัฒนาทักษะการใช้เครื่องมือ Data Analytics และ Visualization  
- เรียนรู้การนำเสนอข้อมูลเชิงวิเคราะห์ในรูปแบบที่เข้าใจง่าย
